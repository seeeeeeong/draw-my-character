## 프로젝트 소개

'AI Service'는 OpenAI의 ChatGPT를 활용하여 이미지 기반 캐릭터 생성 및 변형 서비스를 제공합니다.

### 프로젝트 기능

[서비스 세부 기능](/docs/service_detail.md)

## 기술 스택

- Java 21
- Gradle 
- Spring Boot 3.3.5
- Spring Data JPA
- Spring Security
- QueryDSL 
- PostgreSQL
- SpringAI
- DALL-E 3
